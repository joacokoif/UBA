# Universidad de Buenos Aires (UBA)

Este repositorio centraliza los trabajos prácticos realizados durante mi formación en la **Facultad de Ciencias Exactas y Naturales (FCEN - UBA)**.

## 📂 Organización por Materias

### 📊 Laboratorio de Datos (LDD)
Materia enfocada en el ciclo completo de vida de los datos. Se exploran técnicas de:
*   **Obtención y Limpieza**: Web scraping, uso de APIs y manipulación de datos con pandas.
*   **Análisis Exploratorio (EDA)**: Visualización y detección de patrones.
*   **Modelado**: Introducción a Machine Learning (K-NN, Árboles de Decisión, Clustering).

**Trabajos incluidos:**
*   `tp_0`: Introducción y configuración del entorno.
*   `tp_1_sube_2023_ldd`: Análisis de datos de transporte público (SUBE).
*   `tp_2_fulbo_ldd`: Proyecto de análisis aplicado al fútbol.

### 📐 Algebra Lineal Computacional (ALC)
Estudio de algoritmos para resolver problemas de álgebra lineal a gran escala con enfoque en la eficiencia computacional. Temas clave:
*   **Factorizaciones**: LU, QR, Cholesky y SVD.
*   **Métodos Iterativos**: Resolución de sistemas lineales complejos.
*   **Aplicaciones**: Implementación de algoritmos como **PageRank**.

**Trabajos incluidos:**
*   `pagerank-alc2024`: Implementación y análisis del algoritmo de ranking de búsqueda.
